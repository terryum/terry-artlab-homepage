# I18N_ROUTING.md

> 언어 감지, 전환, fallback 로직만 정의한다.
> URL 경로 구조는 `SITEMAP_IA.md` 참조.

---

## 지원 언어 (v1)
- `ko`, `en`

## 기본 원칙
- 모든 공개 페이지는 언어 prefix 사용: `/ko/...`, `/en/...`
- `/`는 진입점이며, 언어 판별 후 `/ko` 또는 `/en`으로 리다이렉트

## 첫 방문 언어 결정 (우선순위)
1. 사용자가 직접 선택해 저장한 언어 (쿠키/localStorage)
2. 브라우저 언어 (`Accept-Language` / `navigator.language`)
3. 기본값 `en`

판별: 브라우저 언어가 `ko`로 시작하면 `/ko`, 그 외 `/en`

## 언어 전환기 규칙
- 모든 페이지에 노출
- 전환 시: 선택 언어 저장 + 동일 slug의 반대 언어 페이지로 이동
- 저장된 언어는 브라우저 언어보다 우선

## 번역 fallback 규칙
1. **요청 언어 존재** → 정상 렌더링
2. **요청 언어 없음, 다른 언어만 존재** → 존재하는 언어로 리다이렉트 + 안내 배너
3. **두 언어 모두 없음** → 활성 언어 기준 404

## 저장/지속성
- 명시적 선택이 있을 때만 저장
- 저장값이 있으면 브라우저 언어로 덮어쓰지 않음

## v1 비범위
- 브라우저 런타임 자동번역, 지역별 세분 언어, 3개 이상 언어
